# Yolando Demand Gen Dashboard

An interactive dashboard presenting a complete 0→1 growth marketing plan for Yolando, an AI-powered AEO/GEO analytics platform. Built as a portfolio case study demonstrating how to structure and present a demand gen strategy for an early-stage B2B SaaS product.

**[View the live dashboard](https://neeldas1209.github.io/Yolando-Demand-Gen-Dashboard/)**

## What's Inside

The dashboard covers the full demand gen planning process across 12 interactive tabs:

| Tab | What It Covers |
|-----|---------------|
| About This Project | What the dashboard is and how it was built |
| Market Context | AEO market sizing, awareness-action gap, competitive pressure timeline |
| Competitive Landscape | Capability matrix with radar chart, expandable competitor battlecards |
| ICP + Buying Committee | Firmographic, technographic, and behavioral signals with buying committee personas |
| Account Intelligence | Sourcing methods, intelligence signals, and sample Tier 1 accounts |
| Scoring + Tiering | Interactive scoring calculator with weighted signals and tier definitions |
| Messaging Framework | Positioning angles, persona-specific messaging, and objection handlers |
| Programs + Channels | Multi-channel campaign strategy with primary, secondary, and partnership channels |
| Funnel Math + Attribution | Pipeline projections, conversion assumptions, and attribution model |
| 90-Day Plan | Phased execution roadmap with milestones |
| Outreach Packages | Persona-specific outreach sequences and messaging |
| AI Execution Layer | AI-powered workflows integrated into the demand gen process |

## Tech Stack

- Vanilla HTML, CSS, and JavaScript — single self-contained file
- Chart.js for data visualizations
- Plus Jakarta Sans typography
- No server, no build tools, no dependencies — opens in any browser

## Context

This was built as a case study for an early-stage B2B SaaS company in the AEO/GEO analytics space. It demonstrates the ability to build a complete 0→1 growth marketing plan from market analysis through competitive positioning, ICP definition, account scoring, messaging, channel strategy, funnel math, and phased execution.
